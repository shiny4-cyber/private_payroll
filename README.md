## 📖 Project Overview
**Aleo Private Payroll** is a privacy-first blockchain payroll solution. This project leverages **Zero-Knowledge Proofs (ZKP)** to ensure confidential salaries and secure balances.

## 🛠️ Usage Guide
### 1. Setup
git clone https://github.com/shiny4-cyber/private_payroll.git && cd private_payroll && leo build

### 2. Disbursement
leo run pay_employee <EMPLOYEE_ADDRESS> 5000u64

### 3. Split Salary
leo run split_salary '<YOUR_RECORD>' 2000u64
